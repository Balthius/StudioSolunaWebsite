def standardize_col(col):
    return (col - col.mean()) / col.std()

def diff(a,b):
    return a - b
def ratio(t,p):
    return p / t

def pearsonsr(x,y):
    std_x= (x-x.mean())/x.std(ddof=0) 
    std_y= (y-y.mean())/y.std(ddof=0) 
    
    return (std_x * std_y).mean()

def split_keywords(kw_col,threshold):
    allkeywords = []
    for keywordrow in kw_col:
        allkeywords.append(keywordrow)

    keyword_set = set()
    for rows in allkeywords:
        rowlist = rows.split("|")
        for row in rowlist:
            keyword_set.add(row)

    total_keyword_count = dict()
    for word in keyword_set:
        count = kw_col.str.count(word).sum()
        if(count > threshold):
            total_keyword_count[word] = count
        
    return total_keyword_count

kw_total = split_keywords(df['keywords'],3)
kw_popular = split_keywords(highest_1000_voted_movies['keywords'],3)
    
total_keyword_series = pd.Series(kw_total, name='word_count')
total_keyword_series.index.name="keyword"

popular_keyword_series = pd.Series(kw_popular, name='word_count')
popular_keyword_series.index.name="keyword"

keyword_compare_df = pd.concat([total_keyword_series,popular_keyword_series], axis=1)
keyword_compare_df.dropna(inplace=True, axis=0)
keyword_compare_df.columns = ['total', 'popular']
keyword_compare_df['difference'] = keyword_compare_df.apply(lambda diff: diff['total'] - diff['popular'], axis = 1)
keyword_compare_df['ratio'] = keyword_compare_df.apply(lambda ratio: ratio['popular'] / ratio['total'], axis = 1)

top_keywords = keyword_compare_df['ratio'].sort_values(ascending =False).nlargest(50, keep='first')

This Section of code has been created to find all keywords in the dataset, split them, then tally them in a dict.

Next I did the same for the top 1000 movies, creating another dict of counts

Finally I merged the two, removed any values that didn't appear in one or the other(so any that weren't 'popular'); Then two columns were added to the DataFrame, the difference between the popular and total, and a ratio of how often the popular results were found in the total results.

Limitations¶

Though the dataset was relatively large, one of the primary limitations I found was since a lot of my prefered measurements involved budget, and half the values had a budget of 0 that I considered a'unlcean' factor, I had to remove half my dataset.

I would have liked those budgets filled out, and I would also have prefered to have a standardized quantity of movies per year, or perhaps per decade. The rising count per year led to some uncertainty as I was cleaning/exploring the data.
